# Petshop Article Prototype

Static HTML prototype for the updated Petshop.ru article section.

## Pages

- `index.html` - article page
- `articles.html` - all articles list

## Scope

The prototype keeps the existing site shell concept: header, footer, breadcrumbs,
side sections and Jivo-safe spacing. It removes category tabs, badges, bookmarks,
view counters, weekly popular block, pet filters and veterinarian consultation
blocks according to the layout task.
